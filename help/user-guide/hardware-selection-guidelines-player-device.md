# 播放器設備的硬體選擇指南 {#hardware-selection}

以下部分提供了AEM Screens播放器的硬體選擇指南。

## 重要考量 {#important-considerations}

* 始終源 ***商業*** 或 ***工業*** PC播放器和顯示面板或投影儀的等級元件。

* 始終與為數字標牌市場服務的供應商接洽。
* 始終考慮環境因素，如環境溫度和相對濕度。
* 始終查看電源要求和電源調節。
* 仔細審查應用程式所需的效能需求和I/O埠。
