# 播放器裝置的硬體選擇指南 {#hardware-selection}

下節提供AEM Screens播放器的硬體選擇准則。

## 重要考量事項 {#important-considerations}

* 請務必 ***為PC播放********* 器和顯示面板或投影儀提供商業級或工業級元件。

* 與為數位標牌市場服務的廠商永遠互動。
* 始終考慮環境因素，如環境溫度和相對濕度。
* 請隨時查看電源要求和電源調節。
* 仔細審查應用程式所需的效能需求和I/O埠。
