# 播放器設備{#hardware-selection}的硬體選擇指南

下節提供AEM Screens播放器的硬體選擇准則。

## 重要注意事項{#important-considerations}

* 始終為PC播放器和顯示面板或投影儀源&#x200B;***商業***&#x200B;或&#x200B;***工業***&#x200B;等級元件。

* 請務必與提供數位看板市場的廠商互動。
* 始終考慮環境因素，如環境溫度和相對濕度。
* 始終檢查電源要求和電源調節。
* 仔細審查應用程式所需的效能需求和I/O埠。
