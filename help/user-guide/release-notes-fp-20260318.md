# Feature Pack 20260318發行說明 {#release-notes-for-screens-feature-pack}

>[!CAUTION]
>
>Adobe建議您升級至最新版Adobe Experience Manager (AEM 6.5)。 您可以從[這裡](https://experienceleague.adobe.com/zh-hant/docs/experience-manager-65/content/release-notes/release-notes)取得最新版本資訊。
>使用FP11.8搭配SP (servicepack) 22版或更新版本。

## 可用性 {#availability}

AEM Screens已發行AEM 6.5 Feature Pack 11.8。

您可以使用您的Adobe ID，從[軟體發佈入口網站](https://experience.adobe.com/#/downloads/content/software-distribution/en/aem.html)下載AEM Screens 6.5.11.8發行版的最新Feature Pack。 瀏覽至&#x200B;**Adobe Experience Manager**&#x200B;索引標籤並搜尋&#x200B;**Screens**，以取得標題為&#x200B;**AEM 6.5 Screens FP11.8**&#x200B;的最新功能套件。

## 發行日期 {#release-date}

AEM Screens Feature Pack 20260318的發行日期為2026年3月18日。
